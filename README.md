## 8 вариант
## Задание 4.2.8. Написать программу, находящую самое частое отчество в фиксированном списке ФИО.
## Решение:

```
try:
    n = int(input())
    middle_names = {}
    count = 0
    
    for i in range(n):
        fio = input().split()
        if len(fio) >= 3:
            middle_name = fio[2]
            middle_names[middle_name] = middle_names.get(middle_name, 0) + 1
            count += 1
    
    if middle_names:
        print("Самое частое отчество:", sorted(middle_names.items(), key=lambda x: x[1])[-1][0])
    else:
        print("В списке не найдено ни одного человека с отчеством.")
    
    print("В расчете участвовало человек:", count)

except:
    print("Ошибка: количество человек должно быть целым числом.")
```
